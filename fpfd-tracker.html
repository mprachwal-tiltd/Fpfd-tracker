🤖 AUTOMATION & SETUP GUIDE
GOAL: When a dispatch text arrives, automatically log the call AND capture your GPS — zero manual entry. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ OPTION A: ANDROID + TASKER (~$3.50 one-time) ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 1. Install Tasker from Google Play 2. Install AutoLocation plugin (free) 3. Create Profile: TRIGGER: SMS Received FROM [your dept number] ACTIONS: a. Get GPS coords (AutoLocation) b. HTTP POST to Google Sheets webhook: → timestamp, GPS lat/lng, SMS body text 4. Google Apps Script receives the POST, logs row 5. Export that sheet monthly and import CSV here ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ OPTION B: iOS SHORTCUTS (free, built-in) ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ 1. Shortcuts → Automation → New → Message Received 2. From: [your dept dispatch number] 3. Actions: a. Get Current Location b. Get Contents of URL (POST request): → URL: your Google Sheets webhook → Body: Date, Location, Message body Note: iOS 17+ supports fully silent automations. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ ZAPIER NOTE ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ Zapier can't read your phone's SMS directly. Best path if your dept uses Active911, Resgrid, or email-based CAD dispatch — Zapier watches that inbox and logs calls automatically. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ NASSAU COUNTY SMS DISPATCH FORMAT ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ Typical formats this app recognizes: "Signal 10 / 123 Main St, Floral Park" "Sig 9 / Signal 9 Cardiac 45 Oak Ave" "WORKING FIRE Box 4-4 Mutual Aid Req" "Signal 33 request from Mineola" "Signal 9A Ambulance 78 Park Blvd" "Signal 99 / False Alarm 200 Elm St" "Signal 32 General Alarm Hempstead" The parser reads the signal number first, then falls back to keyword matching if no signal number is found. ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ SHARING YOUR REPORT ━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━ Use the REPORT tab to view your activity summary and export a formatted PDF showing your response statistics, availability breakdown, call patterns, and overnight coverage hours.
